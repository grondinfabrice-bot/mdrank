# Guide bêta MDRank

MDRank sert à publier des punchlines courtes, sous pseudo, et à faire monter les meilleures dans les classements.

## Les bases

- Ton email sert seulement à te connecter.
- Ton identité publique est ton pseudo.
- N'utilise pas de vrai nom public.
- Publie court : une bonne punchline doit frapper vite.
- Les votes sont anonymes publiquement.

## Réactions

- 😂 Drôle : réaction positive simple.
- 🤯 Déjanté : absurde, original, inattendu.
- 🔥 Lourd : validation forte.
- 💀 Assassin : meilleure réaction normale.
- 🧊 Pas ouf : existe, mais reste une réaction discrète.

## SuperNote

La SuperNote ⭐ est rare : une fois par jour. Garde-la pour une punchline qui mérite vraiment de monter.

## Règles simples

- Fais rire, ne vise pas une vraie personne.
- Pas d'attaque contre une personne identifiable.
- Pas de vrais noms.
- Signale une punchline si elle pose problème.

## Pendant le test

Si quelque chose semble bizarre, note l'écran, l'action faite, le résultat attendu et le résultat obtenu. Une capture aide beaucoup.
